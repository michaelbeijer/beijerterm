# LLDPE

lineair laagdichtheid polyethyleen; liniaire polyethylenen met lage dichtheid =
 linear low-density polyethylene (abbreviated **LLDPE**)

 lineair heel laag dichtheid polyethyleen = 
 linear very low-density polyethylene (abbreviated **VLDPE**, or sometimes LVLDPE)

Note:

- VLDPE = default, widely accepted, implicitly linear
- LVLDPE = valid but niche, used for clarity or contrast, especially when architecture matters
