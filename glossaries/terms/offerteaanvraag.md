# offerteaanvraag

1. # Dutch
- offerteaanvraag
- offerte-aanvraag

1. # English
- request for a quotation (Van Dale online)
- tender offer (https://iate.europa.eu/search/result/1748789847290/1)
- request for an offer (*een offerteaanvraag indienen = to request an offer*)
- **(aanbest.)** invitation to tender, request for tender (*beperkte offerteaanvraag = restricted invitation to tender, restricted request for tender*)

1. # Gemini

- **Request for Quote (RFQ) –** is often used for simpler procurements where price is the main factor.
- **Request for Proposal (RFP)**<ref name=":0" /> – is used for more complex procurements where the response (the "Offerte" or Offer) needs to include detailed technical information, methodologies, and project plans, not just pricing.
- **Request for Tender(s)** **(RFT)** or **Invitation to Tender (ITT)**<ref name=":0">https://en.wikipedia.org/wiki/Invitation_to_tender</ref> of **Call for Bids** – are also common, particularly in public procurement processes like the "Europese Openbare Aanbesteding" (European Public Tender) mentioned in the Offer Guide.
