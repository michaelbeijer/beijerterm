# wisselmatrijs

- *wisselmatrijs** is a Dutch plastics engineering term with multiple possible meanings depending on context.

1. ## Possible English equivalents
- modular mould system, modular mold system
- quick-change mould system, quick-change mold system
- interchangeable mould insert, interchangeable mold insert
- exchangeable mould system, exchangeable mold system

1. ## Alternative meaning
Some technical dictionaries (such as the GWIT) list **sliding punch** as the translation. This refers to a specific mould component: a movable core insert or slide used to create undercuts or other features perpendicular to the opening direction. In contemporary injection moulding English, this is usually called a **sliding core**, **slide**, or **side action**.

1. ## Choosing the right translation
- If the text concerns flexibility, quick changeovers, sustainability, reduced steel use, or product variants: use *interchangeable mould insert*, *exchangeable/modular mould system*, or *quick-change mould system*.
- If the text concerns mould movements, undercuts, side features, or actuated components within the tool: use *sliding core* (occasionally *sliding punch*).

1. ## Usage notes
In many contexts, *wisselmatrijs* denotes a mould concept in which only cavity inserts are exchanged while the base frame remains. Benefits include lower material use and cost, shorter lead times, and greater flexibility for design changes.

1. ## Example
: *Dankzij wisselmatrijzen hoeven we niet telkens een volledig nieuwe matrijs te bouwen.*
: **Thanks to interchangeable mould systems, we do not need to build a completely new mould each time.**
