# eenzelfde

**NOTE**: I always have trouble with ‘eenzelfde’, which is used differently in Dutch Dutch (nl-NL) and Belgian Dutch (nl-BE).
1. # Dutch
- eenzelfde
- éénzelfde
- een zelfde (According to Handboek Verzorgd Nederlands (Klein/Visser, Martinus Nijhoff) it OUGHT to be spelt as 'een zelfde'.)
1. # English
- the same (luchthavens bij eenzelfde plaats = airports serving the same city) (eenzelfde ervaring = the same experience, **Van Dale**)
- similar
- (gelijk zijnde) identical (Van Dale Online)
- equal (eenzelfde recht hebben om te werken = to have an equal right to work/operate)
- single (Belgian Dutch?)
- (analoog) equivalent, analogous, cognate, corresponding, correspondent (Van Dale Groot woordenboek Nederlands-Engels (versie 2.0, 2002))
1. # Dictionaries
