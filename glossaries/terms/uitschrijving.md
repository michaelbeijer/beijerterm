# uitschrijving

1. # Dutch
- uitschrijving (other forms: uitschrijven, uitgeschreven, etc.)

1. # English
- deregistration
- unsubscribe
- termination of enrollment (See also: uitschrijfverzoek = request for termination of enrollment //  uitschrijven (van student) = terminate the enrollment of…)
- strike off (iemand als lid uitschrijven = strike someone’s name off the membership list)
- delete from
- removal from (a register) (zich laten uitschrijven uit Oldenzaal = have one's name removed from the municipal register at Oldenzaal)
