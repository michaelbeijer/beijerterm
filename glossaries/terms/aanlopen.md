# aanlopen

**See also**: inlopen
1. # Dutch
- aanlopen, aangelopen
1. # English
1. start, initiate 
1. (een haven binnenvaren) call at 
1. (m.b.t. elkaar rakende delen) interfere with, rub against 
1. (m.b.t. wiel) run out of true, be out of true 
1. (schuring van touw) chafe 
1. (van een metaal) tarnish, become oxidized 
1. (verkleuren) colour, tarnish
1. # IATE
 aanlopen van een machine =
 start a machine
 
 **Definition**: to set in motion. This operation generally consists in engaging the motor or motors (Wüster: Machine Tool 1968)
1. # Wolters
 ‘4. rub [agamst]; (ve rem) drag;
 (van een wiel) be out of true, not run true’
 
 **Wolters’ Handwoordenboek Nederland-Engels** (20e druk, 1994)
