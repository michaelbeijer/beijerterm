# egaal

1. # Dutch
; egaal
: **1. gelijkmatig van voorkomen, zonder afwisseling**
::synoniem: effen
: **2. gelijk van oppervlak** (‘een egale massa, vloer’)
::synoniem: effen, glad 
: **3. om het even** (‘’t is mij egaal’)
:: synoniem: onverschillig

1. # English
; egaal 
: smooth, level, even 
: uniform, solid, even [e.d.](kleur)

1. ## Examples
; egaal grijs
: solid / uniformly grey 
; een egale lucht
: a uniform sky
